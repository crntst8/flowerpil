New Playlist: Apple Flow

Getting Started

1. Change account type

open dsp settings →

then change account type →

2. Connect your own account

← open connect apple music

3. Confirm & exit config

confirm it shows ‘using your account’ →

then go back to playlists →

open ‘create playlist’ →

4. Open creator & import

expand the import tools →

select the Apple Music tab and import your playlist →

Making Playlists

Step 1: Track Editor

Once finished, tracks will appear in the editor. You can adjust / add the metadata and drag around to reorder.

Track Quotes

You can add quotes that will ammend on the top of the track if you’d like to provide additional context, thoughts, feelings.

Adding Custom Sources [Bandcamp, Soundlcoud, Youtube etc)

Open the METADATA editor in the track row and navigate to Custom Sources to add links to other platforms of your choosing

Step 2: Playlist Details

Once you’re happy with the track order, metadata and quotes, move on to edit the Playlist Details.

It’s mostly straightfoward things - title, short description (one-liner), full description (rich-text/linebreaks if you want longer text),

There’s some extra bits and bobs such as:

Genres

These are displayed on the homepage links to the playlsits and are added manually.

Genre Hints

Just autocomplete based on what you type - click to fillout.

  👇

Custom Genres

If you want to add your own genre, you can just ignore the hints and add!

  👇

Call-To-Action Button

You can add a link to any platform with your own CTA

Expand the add icon section if you want to use your own brand

No icon will default back to this nice lil flower.

Step 3: Playlist Details

Upload an image to use as the playlist cover!

Step 4: Publish Stage

Publish & Export

Exporting to site + DSPs (Spotify, Apple Music & TIDAL)

Select the platforms you wish to export to and configure which account to use (select Flowerpil if you dont have one)

The links will add themselves back to the playlist upon export.

Publish Only

This will skip the export stage and just publish to the site.

You can go back and export/adjust in the editor after publication.
